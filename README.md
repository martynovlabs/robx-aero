# 3D симулятор квадрокоптера

Веб-приложение для 3D-симуляции полёта квадрокоптера с трассами и редактором.

## Что улучшено для реалистичности

- Физика с массой, гравитацией, тягой и аэродинамическим сопротивлением.
- Наклон по pitch/roll влияет на направление тяги, поэтому коптер летит «как в жизни», а не по осям напрямую.
- Отдельный yaw (`Q/E`) и ограничение максимального угла наклона.
- Инерция: коптер не останавливается мгновенно, есть разгон/торможение.
- Анимированные пропеллеры (скорость зависит от throttle).
- Более реалистичная 3D-модель рамы: корпус, лучи, моторы, шасси.
- Камера преследования с плавным демпфированием.

## Управление

- `WASD` или стрелки — pitch/roll (движение вперёд/назад/влево/вправо через наклон).
- `Space / Shift` — увеличение/уменьшение throttle (подъём/снижение).
- `Q / E` — рыскание (yaw).

## Трассы

- Полёт через кольца по порядку с таймером круга.
- Редактор трассы:
  - **Новая трасса** — включить редактор,
  - клик по сцене — добавить кольцо,
  - `Alt + drag` по кольцу — переместить,
  - **Сохранить трассу** — сохранить в `localStorage`.

## Запуск

```bash
python3 -m http.server 8000
```

Откройте `http://localhost:8000`.

## Как сделать ещё реалистичнее

Если нужно близко к реальному FPV-коптеру:

1. Добавить PID-контроллеры (rate/angle mode).
2. Считать тягу по каждому мотору отдельно и моделировать микшер моторов.
3. Добавить ветровые порывы и турбулентность.
4. Сделать столкновения с объектами и повреждения.
5. Поддержать режим Acro и ограничения по мощности батареи.
