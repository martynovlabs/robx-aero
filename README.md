# 3D реалистичный симулятор квадрокоптера

Симулятор полёта квадрокоптера в 3D с трассами, редактором и приближённой к реальности динамикой.

## Что уже сделано для реалистичности

- **Физика на силах/моментах**, а не простое перемещение позиции.
- Масса, гравитация, тяга от 4 моторов, аэродинамический drag.
- Угловая динамика (roll/pitch/yaw) через моменты и инерцию корпуса.
- Моторный микшер (4 мотора с разными вкладами в roll/pitch/yaw).
- Ограничения тяги (`throttle`) и лёгкая просадка батареи (`battery sag`).
- Реалистичнее 3D-модель: фюзеляж, лучи, моторы, пропеллеры, шасси.
- Визуальная среда (освещение, тени, крупная сцена, препятствия) и плавная chase-камера.
- Телеметрия в UI: скорость, высота, тяга, режим.

## Управление

- `WASD` или стрелки — крен/тангаж (roll/pitch).
- `Q / E` — рыскание (yaw).
- `Space / Shift` — увеличить/уменьшить тягу (throttle).

## Трассы

- Пролёт колец по порядку + таймер круга.
- Редактор:
  - **Новая трасса** — новый маршрут,
  - клик по сцене — добавить кольцо,
  - `Alt + drag` — переместить кольцо,
  - **Сохранить трассу** — сохранить в `localStorage`.

## Запуск

```bash
python3 -m http.server 8000
```

Откройте `http://localhost:8000`.
