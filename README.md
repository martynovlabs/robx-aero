# Симулятор квадрокоптера

Небольшой фронтенд-симулятор квадрокоптера на HTML/CSS/JS.

## Возможности

- Управление коптером с клавиатуры: `WASD` или стрелки.
- Прохождение трассы через ворота по порядку с фиксацией времени круга.
- Создание новой трассы в режиме редактора (добавление ворот кликами).
- Редактирование трассы перетаскиванием ворот мышью.
- Сохранение и загрузка трасс (в `localStorage`).

## Запуск

Откройте `index.html` в браузере или поднимите простой сервер:

```bash
python3 -m http.server 8000
```

После запуска откройте `http://localhost:8000`.
