# 3D симулятор квадрокоптера

Веб-приложение для 3D-симуляции полёта квадрокоптера с управлением с клавиатуры и редактором трасс.

## Возможности

- 3D-сцена на Three.js (коптер, освещение, сетка, трасса из колец).
- Управление полётом:
  - `WASD` или стрелки — вперёд/назад/влево/вправо,
  - `Space` / `Shift` — подъём/снижение.
- Прохождение трассы по кольцам в заданном порядке с замером времени круга.
- Редактор 3D-трассы:
  - кнопка **Новая трасса** включает режим редактирования,
  - клик по сцене добавляет кольцо,
  - `Alt + drag` по кольцу перемещает его,
  - кнопка **Сохранить трассу** сохраняет трассу в `localStorage`.
- Загрузка ранее сохранённых трасс из списка.

## Запуск

```bash
python3 -m http.server 8000
```

Откройте `http://localhost:8000`.
